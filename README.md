# Uppgift 2 - Frontendbaserad webbutveckling

Projektet omfattar utveckling av en webbapplikation för visning och filtrering av ett ramschema. Applikationen är byggd med Vite och Vanilla JavaScript.

## Live-version
Webbplatsen finns publicerad på:
https://runegranath.github.io/Uppgift-2-Frontendbaserad-webbutveckling/

## Funktionalitet
- Hämtning av schemadata från JSON-format.
- Sortering av kurslista (kurskod, kursnamn och poäng).
- Sökfilter för dynamisk filtrering av tabellinnehåll.
